・基本的な使い方

using KZ.IOLib　を入れる必要あり

・csvの読み込み
CSVIO.LoadMap()メソッドに
マップを読み出す配列(現在はint,byte,string,任意のenum型に対応),
読み込むCSVファイルのパスまたはUnityのTextAssetとしてCSVファイルを渡す
(IOTester.csでの処理を参考にしてください)

追加機能：
LoadMap()関数に読み込まず飛ばす文字列およびファイルの終端を表す文字列を指定できるようにしました。
（読み込みを飛ばす文字列はその文字列が含まれているセルの読み込みがスキップされます。
また、ファイルの終端文字列以降のセルは読み込みされません）

→現在このオプションを使用できるのはstring型の2次元Listに読み出す場合のみですが
2次元配列にも対応する予定です。

・csvへの書き出し
CSVIO.SaveMap()メソッドに書き出す要素の入った2次元配列,書き込み先のパスを引数に指定
パスの指定などはIOTester.csのStart()関数およびTestSave()関数を参考に

→TextAssetに書き込み先を指定できるように拡張する予定？
