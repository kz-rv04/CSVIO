2018/01/17 更新

・基本的な使い方

using KZ.IOLib　を入れる必要あり

・csvの読み込み
CSVIO.LoadMap()メソッドにマップデータを読み込む配列やリストを渡すことでCSVファイルの内容を読み込むことができます。
(IConvertable インターフェースを実装する型に対応)


・listに読み込む場合は読み込まないセルを指定することができます
IOTester.csのようにList ignoreに読み込みを無視する文字列を指定することで
指定した文字列を含むセルを読み込まないようにできます。

ファイルの終端子(EOF)も引数で指定することができます(指定しない場合はファイル末尾まで読み込み)

読み込むCSVファイルのパスまたはUnityのTextAssetとしてCSVファイルを渡す
(IOTester.csでの処理を参考にしてください)

・csvへの書き出し
CSVIO.SaveMap()メソッドに書き出す要素の入った2次元配列,書き込み先のパスを引数に指定
こちらも読み込み時と同様に基本型や列挙型に対応
